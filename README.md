# Task Manager Frontend

React frontend for the Django Task Manager backend.

## Setup

1. Open a terminal in `frontend`
2. Run `npm install`
3. Run `npm run dev`

The app expects the backend API to be available at `http://127.0.0.1:8000`.

## Features

- Login / register with JWT auth
- List tasks assigned to or created by the user
- Create, edit, and delete tasks
- Simple Vite-powered React app with proxy configuration
